# Lantai リリース

[中文](README.md) | [English](README_en.md) | [한국어](README_ko.md)

この公開リポジトリは、Lantai デスクトップアプリのリリース資産、GitHub Releases、自動更新メタデータを置くためのものです。

ソースコードのリポジトリは非公開です。ユーザーはこのリポジトリの Releases ページからインストーラを入手するか、デスクトップアプリのバックグラウンド更新を利用できます。

## ダウンロード

- [Releases](https://github.com/shenyangs/lantai3-releases/releases)

## 自動更新

Lantai デスクトップアプリはバックグラウンドで静かに更新を確認します。新しいバージョンが見つかると静かにダウンロードし、次回アプリを通常終了したときにインストールします。

## ファイル

- `latest-mac.yml`: macOS 自動更新メタデータ
- `*.dmg`, `*.zip`: macOS インストーラと更新資産
- `latest.yml`, `*.exe`: Windows インストーラと更新資産
