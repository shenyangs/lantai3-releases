# Lantai 릴리스

[中文](README.md) | [English](README_en.md) | [日本語](README_ja.md)

이 공개 저장소는 Lantai 데스크톱 앱의 릴리스 파일, GitHub Releases, 자동 업데이트 메타데이터를 보관합니다.

소스 저장소는 비공개로 유지됩니다. 사용자는 이 저장소의 Releases 페이지에서 설치 파일을 내려받거나, 데스크톱 앱의 백그라운드 자동 업데이트를 사용할 수 있습니다.

## 다운로드

- [Releases](https://github.com/shenyangs/lantai3-releases/releases)

## 자동 업데이트

Lantai 데스크톱 앱은 백그라운드에서 조용히 업데이트를 확인합니다. 새 버전이 있으면 조용히 다운로드하고, 앱을 정상 종료한 다음 번에 설치합니다.

## 파일

- `latest-mac.yml`: macOS 자동 업데이트 메타데이터
- `*.dmg`, `*.zip`: macOS 설치 및 업데이트 파일
- `latest.yml`, `*.exe`: Windows 설치 및 업데이트 파일
