# Lantai Releases

[中文](README.md) | [日本語](README_ja.md) | [한국어](README_ko.md)

This public repository hosts Lantai desktop release assets, GitHub Releases, and auto-update metadata.

The source repository remains private. End users can download installers from this repository's Releases page or let the desktop app update itself in the background.

## Downloads

- [Releases](https://github.com/shenyangs/lantai3-releases/releases)

## Auto Updates

The Lantai desktop app checks for updates quietly in the background. When a new version is available, it downloads the update silently and installs it the next time the app exits normally.

## Files

- `latest-mac.yml`: macOS auto-update metadata
- `*.dmg`, `*.zip`: macOS installer and update assets
- `latest.yml`, `*.exe`: Windows installer and update assets
